Первая вкладка:
"Валюта 1" - выберите из выпадающего списка валюту первого вклада.
"Сумма вклада" - введите сумму вклада в выбранной валюте. Допускается ввод только целых чисел.
"Годовая % ставка" - укажите годовую процентную ставку вашего банка по выбранному вкладу.
"Капитализация" - выберите вид капитализации, если таковой предусмотрен Вашим вкладом.
Доступны ежедневная, ежемесячная (каждый 30-ый день), квартальная (каждый 90-ый день) и годовая капитализации.
"Срок" - служит для ввода продолжительности вклада. Допускается ввод только целых чисел.
"Дата начала" - введите дату начала вклада. Это поле не влияет на вычисления и используется для расчета окончания вклада.
"Дата окончания" - рассчитывается автоматически на основании данных из полей "Срок" и "Дата начала".
"Прирост %" - показывает на сколько процентов итоговая сумма в конце срока превышает начальную.
"Прибыль" - денежные средства, принесенные вкладом.
"Итого на вкладе" - итоговая сумма на счету по окончании срока вклада.

Вторая вкладка:
Значение одноименных полей в ней аналогично первой вкладке.
Данные полей "Сумма вклада"*, "Срок", "Дата начала" и "Дата окончания" переносятся с первой вкладки и не могут быть отредактированы.
Поля, доступныe для ввода даных:
"Валюта 2" - выберите валюту второго вклада.
"Конверсия" - установите более удобное направление конверсии валют.
"Кросс-курс" - отношение между единицей валюты первого вклада и единицей валюты второго (в зависимости от направления конверсии).
Пример утановки конверсии и кросс-курса:
Предположим валюта первого вклада 1 USD равен 50 RUR валюты второго. Если утановить конверсию как USD/RUR, то кросс-курс будет равен 50.
Для этой же пары валют можно установить направление конверсии как RUR/USD. В этом случае кросс-курс следует задать как 1/50=0,02.

 *"Сумма вклада" - рассчитывается автоматически на основе суммы первого вклада и величины кросс-курса.

 Третья вкладка содержит сравнение двух депозитов. Процентные величины рассчитываются относительно первого вклада.
 Секция "Итоговые суммы равны при курсе" показывает кросс-курс валют, при котором итоговые суммы обоих вкладов одинаковы.
 В скобках указано его изменение относительно введенного на второй вкладке кросс-курса.

 Секция "Сравнения вкладов при заданном обменном курсе" вычисляет какой вклад выгоднее при курсе заданном пользователем.
 Курс задается как непосредственно вводом в поле "Выбрать обменный курс", так и с помощью ползунка ниже этого поля.
 Таблица "Сравнение прибылей" рассчитывает прибыли каждого вклада в обоих валютах и вычисляет разность между ними. Отрицательная разность, означает что второй вклад приносит больше прибыли.
 Таблица "Сравнение итоговых сумм" считает аналогичную статистику для итоговых сумм вкладов. По ее значениям принимается решение о том какой вклад более выгодный.
